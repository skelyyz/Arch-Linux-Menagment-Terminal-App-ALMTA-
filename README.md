This application is made specificly for myself and people who are in need for proper maintaining of their system without having to write tons of different commands.

----- INSTRUCTIONS -----

You must firstly go to: nano ~/.local/bin/arch-maintenance

then: paste the code.txt file there

Save as:
~/.local/bin/arch-management

Make executable:
chmod +x ~/.local/bin/arch-management

Run:
~/.local/bin/arch-management
